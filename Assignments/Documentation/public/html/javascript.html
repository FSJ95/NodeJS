JavaScript

If you wanna do anything cool on your website, JavaScript is your guy :)

JavaScript is a 'programming language' made for web, its able to change/update HTML/CSS and calculate, manipulate and validate data.

Variables

JavaScript variables are containers for storing data values.

There is multiple ways to define a variable:

const: Const is a constant and cant be changed/reassigned.

let: May be reassigned/changed fx. in a loop. (it is block scoped)

var: Can be eassigned/changed (it is function scoped)

Datatypes

Some of the most used datatypes:

Objects

JS objects is simply a hashmap with key-value pairs.

The value can be any datatype (yes, even functions :D).

Strings

Strings can be written with double and single quotes.

Numbers

JS has only one type of number. In other languages you have datatypes like 'float' and 'int',
but in JS numbers can be written with or without decimals.

Functions

This function returns the first value plus the second value.
This is even possible to add strings.

Fun facts:

JS can run even if you forget to put a semicolon after a statement, but it is good practice to always put them.

You can empty a variable in JS by setting it as 'null'.

This is not to be confused with a 'undefined' object, which means the object was never defined (surprise :D).

Using == dont compare types.

Using === also compare the object types.
