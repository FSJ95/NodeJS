Cascading Style Sheets (CSS)

CSS stands for 'Cascading Style Sheets' and is used to 'describe' how HTML elements are displayed.

There is multiple ways to include CSS in your HTML file:

Inline

It can be written directly on the HTML element you want to change,

External file

it can be imported in the header as a seperate file

Internal

or it can be direcyly written in the header with the <style> tag.

Cascading Order

If multiple sources are used, it will cascade into following order:

Inline styles → External/Internal → Browser default
