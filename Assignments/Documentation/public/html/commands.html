Commands

Here are some useful commands:

Unix commands

cd [directory]: Change working directory to specified.

ls: List all files in working directory.

rm [filename]: Remove specified file.

rm -rf [directoryname]: Remove specified directory.

history: Displays last used unix commands.

mkdir [directoryname]: Create directory with specified name.

echo "test" > test.txt: Create file with specified contents and name.

Git commands

git init: Initializes github in current directory.

git pull: Pulls changes from remote

git add A: Adds all changes to commit

git commit -m "message": Commit added changes with a message.

git push: Pushes commit to remote

git status: Check what's changed since last commit.

git clone [gitrepo]: Pulls gitrepo to current directory.

NodeJS/NPM commands

npm install: Installs all packages from the 'packages.json' file.

npm install -g npm@latest: Install/Update to latest npm version.

npm install [package]: Install local package

npx: Npx used to run packages once.

node [appname]: Run node project once.

node --version: Display current node version.
