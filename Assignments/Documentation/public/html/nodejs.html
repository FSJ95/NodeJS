NodeJs

Wow JavaScript is so cool, i wish i could use it for something else than HTML...

Well it's your lucky day! NodeJS is made with the Chrone v8 JS engine and allows to run JavaScript outside the browser.

Cool thing is it works on most major platforms: iOS, Windows, Linux & Unix.

What to use Node for

Since Node is ideal for building highly scalable, data-intensive backend & real-time apps
you can basicly use it for everything. Here is some om the most common projects:

  • API's
  • Websites/Webservers
  • CMD Programmes
  • Scripts

Packages

Packages is one of the most essential features of NodeJS. It allows to quickly implement tools and libaries to use with your projects.

Packages is installed using NPM, who also provides a huge database/collection of node packages.

Packages is imported using the require("example") method.

Useful packages

Here is some packages i've found very useful:

Express

Express is an very fast and lightweight web framework for Node

Express allows you to build webservers, api's and much more.

This site is build using express.

Nodemon

Nodemon is a utility that will monitor for any changes in your source and automatically restart your server.
