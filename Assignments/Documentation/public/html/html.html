HyperText Markup Language (HTML)

HTML stands for 'HyperText Markup Language' and is the standard language for documents designed to be displayed in a web browser.

In html you use different tags depending on what kind of information you want to serve the browser.

The browser then interpret the tags and displays the content of the page.

Simple html page

Heres a very simple html page:

  • <!DOCTYPE html>
  • Is the declaration that defines this document to be HTML5
  • <html>
  • Is the root element of an HTML page
  • <head>
  • Is the element that contains meta information about the document
  • <title>
  • Is the element that specifies a title for the document
  • <body>
  • Is the element that contains the visible page content
  • <h1>
  • Is the element that defines a large heading
  • <p>
  • Is the element that defines a paragraph
